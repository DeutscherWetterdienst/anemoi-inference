Installing
==========

To install the package, you can use the following command:

.. code-block:: bash

    pip install anemoi-inference

Contributing
------------

.. code-block:: bash

    git clone ...
    cd anemoi-inference
    pip install .[dev]
    pip install -r docs/requirements.txt

You may also have to install pandoc on MacOS:

.. code-block:: bash

    brew install pandoc
